# HotelOccupancy
The program asks the user for how many floors a hotel has, then asks 
how many rooms per floor and how many of those rooms are occupied. Once
all the information has been entered the program outputs to the user a 
percentage of occupied rooms to the user.

# Required Software
Visual Studio 2017

# Instructions
Clone or download the program. Open the project in Visual Studio 2017.
Build the project and then start without debugging to run the program.
